# fonts-cdn
Making a locally downloaded font available to online use via CDN 

<> Usage<>
@font-face {
    font-family: 'Nesto-Copper42-2765';
    src: url('https://sushanth-hebri.github.io/fonts-cdn/Nesto-Copper42-2765.woff') format('woff');    
    font-weight: normal;
    font-style: normal;
}
@font-face {
    font-family: 'TheGlobePersonalUseBold-2ORlw'; /* Replace with your font name */
    src: url('https://sushanth-hebri.github.io/fonts-cdn/TheGlobePersonalUseBold-2ORlw.ttf') format('truetype'); 
    font-weight: bold;
    }

    
    .box{
    font-family: 'TheGlobePersonalUseBold-2ORlw' !important;
    }
